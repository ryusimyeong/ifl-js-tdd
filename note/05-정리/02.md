# 2. 정리

자바스크립트는 런타임에 에러를 발생시키기 때문에 자바스크립트로 개발할 때는 독특한 마음가짐을 가져야 한다. 

컴파일러가 없는 한 테스트가 안전한 개발을 하기 위한 최선의 방법이다.

테스트주도개발 TDD는 `테스트 실패 -> 테스트 성공 -> 리팩토링`의 순서로 진행되며

테스트 코드는 `준비 -> 실행 -> 검증` 순서로 작성한다.

테스트할 수 있는 좋은 코드는 SOLID하고 DRY한 코드다.
